# simple page

코딩 연습을 위해 만든 샘플 포트폴리오 사이트입니다.
100% 반응형으로 제작되어 모니터, 타블렛, 휴대폰 사이트 모두 반응하도록 제작되었습니다.

온라인상에 배포되는 free template를 사용했습니다.
아래의 샘플페이지에서 확인할 수 있습니다.
https://baby-lamb.github.io/simplepage-html-jquery/
